#### 栈的概念

`结合下面的代码片段，分析栈的执行过程：`

```js
  <script>
    function bar () {
      console.log('bar')
    }

    function foo () {
      console.log('foo')
      bar()
    }

    foo()
  </script>
  
```

![栈的执行过程](http://image.yalingmai.cn/zhan.png)

> 从入栈出栈的过程可以看出：js先执行同步任务。遇到同步任务中异步函数先丢到异步模块，再继续执行下面的同步任务。任务执行完成就出栈。当同步任务全部执行完成后，才会去搜索异步任务，按顺序排队入栈执行。
